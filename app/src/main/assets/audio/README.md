# Lingqi audio assets

Guided non-478 sessions include the quiet-looping Pixabay track
`meditation-relax-sleep-music-346733.mp3`. The original MP3 was downloaded from
the official asset page on 2026-07-11 and is kept unchanged; its SHA-256,
creator, source, and public license evidence are recorded in `licenses.json`.

The device's Chinese text-to-speech engine supplies guided speech, so voice
availability still depends on the installed Android TTS voice.

The default 478 cues are the short `pendulum_tick.wav` and
`pendulum_tock.wav` excerpts derived from Universfield's “Old Clock Ticking”
on the official Pixabay page. The original 13.752-second MP3 stays outside the
repository and APK; exact source, license, trim points, processing, and hashes
are recorded in `licenses.json`. The procedurally generated bell cues remain
available as a user choice. Neither cue mode mixes with the ambient track.
