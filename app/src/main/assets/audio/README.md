# Lingqi audio assets

Guided non-478 sessions include the quiet-looping Pixabay track
`meditation-relax-sleep-music-346733.mp3`. The original MP3 was downloaded from
the official asset page on 2026-07-11 and is kept unchanged; its SHA-256,
creator, source, and public license evidence are recorded in `licenses.json`.

The device's Chinese text-to-speech engine supplies guided speech, so voice
availability still depends on the installed Android TTS voice. The 478 cues are
synthesized at runtime and deliberately do not mix with the ambient track.
